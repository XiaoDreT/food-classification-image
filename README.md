
# Food Image Classification

This project implements a machine learning model for classifying food images into different categories.


## Project Specifications

 - Task: Multi-Class Image Classification
 - Classes (6 Classes): Baked Potato, Crispy Chicken, Donut, Sandwich, Taco, Taquito, Fries 
 - Dataset: [Food Image Classification Dataset](https://www.kaggle.com/datasets/harishkumardatalab/food-image-classification-dataset)
 - Model: Convolutional Neural Network (CNN)
 - Framework: TensorFlow, Keras, Seaborn, Numpy, Pandas, Matplotlib
